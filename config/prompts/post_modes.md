# 投稿モード定義

週次戦略 (ADR-012) に基づく 2 モード。曜日推奨はあくまで**補助**、妻が手動で選択できる。

---

## preparation — 準備期間 (月〜木推奨)

**目的**: 信用貯金フェーズ。アルゴリズムに「よく喋る人間」と覚えさせる。

### ルール
- 商品紹介**禁止**
- アフィリンク**禁止**
- 「魂の 5 大お題」 (`config/prompts/soul_topics.md`) のどれか 1 つから書く
- `#PR` なし
- タグは 1 つだけ (ジャンル系: `#時短家事` `#子育て` `#日々のこと` 等)
- 質問で締める
- ゴールデンタイム (朝 7-9 / 昼 12-13 / 夜 20-22) 投稿推奨
- 投稿間隔 **2 分以上**

### UI ヒント
- 商品 URL 入力欄は無効化 or 非表示
- 魂の 5 大お題セレクタを表示
- 「このモードは**リンクなし**です」と明示表示

---

## affiliate — 本投稿 (金〜日推奨)

**目的**: 週末の買い物欲ピークで、2 段階投稿によりアフィ収益を取る。

### ルール
- 商品 URL を取得 → アフィリンク発行
- **親ポストには本文のみ** (URL 入れない — 初期 15 分の露出に致命傷)
- 自動で自己リプライを作成し、そこにアフィリンクを配置 (Stage 3 で実装)
- `#PR` 必須
- タグは本文末尾に 1 つ + `#PR` (計 2 つまで許容、ジャンルタグ 1 + #PR 1)
- 会話誘発の問いかけ必須
- 投稿後 15 分以内に自己リプライ返信 or リプ返信を推奨
- 投稿間隔 2 分以上

### UI ヒント
- 商品 URL 入力 → 商品情報表示 → プロンプト組み立て → 生成
- 生成文に URL が混入していないことを**バリデーション** (Stage 4 で実装)
- 「投稿後 15 分以内に 1 回返信しよう」トースト (Stage 3 で表示)

---

## デフォルト選択ロジック (v0.1、UI 側)

```typescript
function suggestMode(date: Date): PostMode {
  const dow = date.getDay(); // 0=Sun, 1=Mon, ..., 6=Sat
  // 準備: Mon(1) Tue(2) Wed(3) Thu(4)
  if (dow >= 1 && dow <= 4) return "preparation";
  // 本投稿: Fri(5) Sat(6) Sun(0)
  return "affiliate";
}
```

妻がウィンドウを開いた時点で曜日から推定。手動で覆せる。
