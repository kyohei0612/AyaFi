# プルダウン ①: フック形式 (導入の型)

"指定なし" の場合は global.md のデフォに従う。

## casual_empathy (共感)

```
「こういうの、わかる人いる?」
「〇〇で悩んでたんだけど、これで解決した」
```
適合: Threads / Bluesky / 柔らかい話題

## number_shock (数字で掴む)

```
「◯◯を 3 ヶ月続けた結果…」
「買ってよかったもの TOP 3」
「1 日 5 分でこうなった」
```
適合: note / Threads / 具体成果ある商品

## question (質問)

```
「◯◯で困ってません?」
「これ、知ってました?」
```
適合: 全 SNS / 考えさせる商品

## story (体験談)

```
「先月こんなことがあって…」
「もう無理と思ってた時に出会ったのがこれ」
```
適合: note / 感情訴求が効く商品

## news (ニュース風)

```
「最近 SNS で話題の◯◯、試してみた」
「あの◯◯が新しくなってた」
```
適合: Threads / Bluesky / トレンド商品
