# 魂の 5 大お題 (準備期間投稿用)

Threads の「おすすめ」フィードに載せる信用貯金を作るための、商品紹介を**含まない**
投稿テーマ集。月〜木に使う (詳細: [ADR-012](../../docs/decisions/012-threads-algorithm-strategy.md))。

LLM は必ずこの 5 つのどれか 1 つのテーマで書く。毎回同じお題を使っても OK だが、
**切り口を変える**こと (同じ「後悔」でも対象商品/対象体験を変える)。

---

## 1. regret — 後悔

最近買って後悔したもの、やらなくてよかったこと、行かなきゃよかった場所。

- 具体的な「金額」や「時間」を入れると共感が強い
- 商品名は出さない (アフィじゃないと伝わらせる)
- 失敗を責めるトーンより「まあ、学び」系の方が Threads で好まれる

例:
- 「3,000 円出したあの折りたたみ傘、結局 3 回使ってお蔵入り。みんなの『コレは買って損した』ってある?」
- 「時短を期待した◯系グッズ、セッティングの方が時間かかった件」

## 2. surprise — 驚き

意外と知らなかった発見、常識が覆った瞬間、「え、みんな知ってた?」系。

- 家事ハックや生活知識だと共感しやすい
- 「私だけ?」と添えると返信が付きやすい

例:
- 「洗濯ネット、用途別に 3 種類使うと時短になるって最近知った。私だけ?」
- 「実は◯◯、電子レンジ OK だったみたい…常識?」

## 3. small_hack — 日常の小さな工夫

毎日やってる時短テク、続けているルーティン、家事 / 子育ての細かい工夫。

- 具体性が命 (何を、いつ、どのくらい節約できたか)
- 「完璧じゃなくていい」ニュアンスが妻のキャラに合う

例:
- 「朝の支度、前夜にアレを用意しておくだけで 10 分浮く。みんなのルーティンも教えて?」
- 「買い物リスト、メモじゃなくて写真で撮る派になったら買い忘れが 0 件に」

## 4. fail — 失敗談

笑える / 共感できる失敗、キッチンでの大惨事、うっかり話。

- 深刻すぎない、自虐すぎない、「あるある」テイストで
- オチに問いかけ or 「みんなもある?」系

例:
- 「冷蔵庫の奥で発見した、買ったこと忘れてた◯◯。みんなの『行方不明食材』は?」
- 「タイマーセットしたはずが…最終的に焦げた話」

## 5. seasonal — 季節もの

今の時期に感じること / やりたいこと / 準備してること。

- 季節感を出す (桜 / 梅雨 / 夏休み / 敬老の日 / クリスマス / 年末等)
- 「そろそろ準備?」「今年はどうしてる?」系の問いかけと相性がいい

例:
- 「そろそろ衣替え、去年サボったツケが今年まわってきた。みんな何月にやる?」
- 「今年の夏、去年より何か 1 つだけ変えたいこと…何にしよう?」

---

## 共通ルール (LLM 向け)

- **質問で締める** (会話誘発必須)
- **商品名 / ブランド名 / アフィリンク絶対なし**
- **ハッシュタグ 1 つだけ** (例: `#時短家事` `#子育て` `#日々のこと`)
- 冒頭 2 行でシーンを立ち上げる
- 絵文字 2-4 個
- 400-500 字に収める
- 妻 (30 代主婦) の 1 人称「私」で
- `[[ここを入力: 具体的なシーン]]` の空欄を残してもよい (妻が最終編集する前提)
