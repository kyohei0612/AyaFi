# プルダウン ②: トーン (語り手のキャラクター)

"指定なし" の場合は各 SNS の md に従う。

## casual (カジュアル)

- 「〜だよ」「〜なんだ」「マジで」「めっちゃ」
- 友達に LINE で話す感じ
- 絵文字やや多め

## polite (丁寧)

- 「〜です」「〜ます」「〜だと思います」
- ブログ記事らしい落ち着き
- note にマッチ

## friendly (フレンドリー中庸)

- 「〜だね」「〜かな」「だから◯◯」
- 親しみ + 程よい距離感
- Threads / Bluesky 両対応

## expert (専門家風)

- 事実ベース、根拠や数値を出す
- 「実際に比較してみると〜」「一般的には〜ですが、本商品は〜」
- テック系 / ガジェット / 健康食品には向かない (薬機法注意)

## mom_friend (ママ友風)

- 「うちの子が〜」「これ、ほんっとに助かる」
- 具体的な育児シーンを織り込む
- 子育て / 時短 / 食品に強い
